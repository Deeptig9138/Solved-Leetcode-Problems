## Problem Statement

Given a string `text`, determine the maximum number of instances of the word:

```text
"balloon"
```

that can be formed using the characters in `text`.

Each character in `text` can be used at most once.

Return the maximum number of complete `"balloon"` words that can be formed.

```text

```

and repeatedly try to construct it from the given string.

Whenever a character is used, I would mark or "strike out" that character so it cannot be reused.

For example:

```text
loonbalxballpoon
```

First formation:

```text

```

Count:

```text
1
```

Use another set of available characters:

```text

```

```text

```

No more complete words can be formed.

Answer:

```text

```

This approach works conceptually but involves repeatedly searching and marking characters.

There is a much simpler observation.

---

## Key Observation

To form one word:

```text
balloon
```

we need:

| Character | Required Count |
| --------- | -------------- |
| b         | 1              |
| a         | 1              |
| l         | 2              |
| o         | 2              |
| n         | 1              |

Instead of constructing the word repeatedly, we can count the frequency of each character in the input string.

The number of balloons we can form depends on the character that runs out first.

---

## Approach

### Frequency Counting

1. Count the frequency of every character in `text`.
2. Determine how many times each required character can contribute:

   * `b`
   * `a`
   * `l / 2`
   * `o / 2`
   * `n`
3. The answer is the minimum of these values.

---

## Dry Run

### Input

```text
text = "loonbalxballpoon"
```

Character frequencies:

```text
b = 2
a = 2
l = 4
o = 4
n = 2
```

Possible balloons from each character:

```text
b → 2/1 = 2
a → 2/1 = 2
l → 4/2 = 2
o → 4/2 = 2
n → 2/1 = 2
```

Take the minimum:

```text
min(2,2,2,2,2) = 2
```

Answer:

```text
2
```

## Solution

```cpp
class Solution {
public:
    int maxNumberOfBalloons(string text) {

        unordered_map<char, int> freq;

        for(char ch : text)
            freq[ch]++;

        return min({
            freq['b'],
            freq['a'],
            freq['l'] / 2,
            freq['o'] / 2,
            freq['n']
        });
    }
};
```

## Complexity Analysis

| Metric           | Complexity |
| ---------------- | ---------- |
| Time Complexity  | O(n)       |
| Space Complexity | O(1)       |

where `n` is the length of the string.

The space complexity is considered O(1) because only lowercase English letters exist.

---

## Why Divide by 2 for 'l' and 'o'?

One balloon requires:

```text
balloon
```

Notice:

```text
l appears twice
o appears twice
```

Therefore:

```text
available balloons from l = freq['l'] / 2
available balloons from o = freq['o'] / 2
```

Example:

```text
freq['l'] = 5
```

Then:

```text
5 / 2 = 2
```

Only 2 complete pairs of `l` exist.


### Why Minimum?

The character with the smallest available count determines how many complete balloons can be formed.

Example:

```text
b = 10
a = 10
l = 2
o = 10
n = 10
```

Even though most letters are abundant:

```text
l/2 = 1
```

So only:

```text
1 balloon
```

can be formed.

---

### Interview Takeaway

Whenever a problem asks:

```text
How many times can I form a word?
```

think:

```text
Frequency Counting
```

instead of actually constructing the word.

The answer is usually determined by the character that becomes the bottleneck.
