## Problem Statement

You are given the heads of two sorted linked lists `list1` and `list2`.

Merge the two lists into one sorted linked list.

The merged list should be made by splicing together the nodes of the original two lists.

Return the head of the merged linked list.

## My Understanding

My initial thought was to treat one list as the "main" or "ideal" list and insert elements from the second list into their correct positions.

For example:

```text id="f97hjq"
List1 = 1 → 2 → 3
List2 = 1 → 3 → 4
```

Take List1 as the base:

```text id="k85uv7"
1 → 2 → 3
```

Then insert elements from List2 wherever they belong:

```text id="zh0jtt"
1 → 1 → 2 → 3 → 3 → 4
```

This approach would work, but every insertion may require searching for the correct position, making the implementation more complicated.

Since both lists are already sorted, there is a more natural solution.

---

## Key Observation

Both lists are sorted.

That means:

```text id="6yt1yb"
The smallest remaining element must be
either the current node of List1
or the current node of List2.
```

So instead of searching for insertion positions, we can simply compare the current nodes and take the smaller one.

This is exactly the same idea used in the merge step of Merge Sort.

---

## Approach

### Two Pointer Merge

1. Create a dummy node to serve as the start of the merged list.
2. Maintain a pointer called `tail` representing the last node of the merged list.
3. Compare the current nodes of both lists:

   * Attach the smaller node to `tail`.
   * Move that list's pointer forward.
4. Move `tail` forward.
5. Repeat until one list becomes empty.
6. Attach the remaining nodes of the non-empty list.
7. Return `dummy.next`.

### Why Use a Dummy Node?

Without a dummy node, we would need special handling for the first node of the merged list.

Using:

```cpp id="rwh4ee"
ListNode dummy(0);
```

allows us to build the list uniformly from start to finish.

At the end:

```cpp id="30ul6z"
return dummy.next;
```

returns the actual head.

## Solution

```cpp id="0d7hsa"
class Solution {
public:
    ListNode* mergeTwoLists(ListNode* list1, ListNode* list2) {

        ListNode dummy(0);
        ListNode* tail = &dummy;

        while(list1 && list2) {

            if(list1->val <= list2->val) {
                tail->next = list1;
                list1 = list1->next;
            }
            else {
                tail->next = list2;
                list2 = list2->next;
            }

            tail = tail->next;
        }

        if(list1)
            tail->next = list1;
        else
            tail->next = list2;

        return dummy.next;
    }
};
```

* `m` = length of List2

## Revision Notes

* My first idea was inserting nodes from one list into another.
* Better approach: merge while traversing both lists simultaneously.
* Always compare the current nodes.
* Attach the smaller node and move its pointer forward.
* Use a dummy node to avoid special cases.
